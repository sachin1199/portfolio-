The short version

I'm Sachin Singh, a Data Science postgraduate student who builds full-stack web applications — not just interfaces, but the systems underneath them: APIs, databases, authentication, real-time logic.

I learn by building. Rather than working through theory first, I tend to start with a real problem — a multiplayer game that needs to sync instantly, an AI that needs to evaluate positions efficiently — and build toward a working solution, picking up the underlying concepts along the way.

That approach shows in the work: a chess engine using alpha-beta pruning to search game trees efficiently, a real-time sync layer for multiplayer, a JWT-secured API built around proper access control. These aren't tutorial projects — they're solutions to specific, hard problems.
